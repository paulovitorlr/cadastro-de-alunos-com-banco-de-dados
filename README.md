# Portal do Aluno

Projeto em C# (.NET) com PostgreSQL e Dapper.

## Tecnologias
- .NET
- PostgreSQL
- Dapper

## Como rodar o projeto
1. Clone o repositório
2. Crie o banco de dados
3. Execute o script `database/create_tables.sql`
4. Copie `appsettings.example.json` para `appsettings.json`
5. Configure a string de conexão
6. Rode o projeto
